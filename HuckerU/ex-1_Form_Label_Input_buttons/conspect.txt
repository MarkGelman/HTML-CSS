1. dir="rtl" -- меняет направление текста с права на лево. Это свойство долджно быть определено в теге html
2. text-align -- свойство, которое распологает текст в теге по введённому в него значению(center....)
3. br -- это тег, который вставляется или между тегами или в самом теге и выполняет функцию клавиши
         Enter на клавиатуре (перевод строки)
4. &nbsp -- специальное выражение, которое используется для вставления в текст пробелов
5. form --  служит для создания документов
6.  label -- используется к примеру в полях ввода(Input), для названия поля
            если значение его равно submit, то на листе это отображается ввиде кнопки "Отправить"
7.  type -- используется в полях ввода. Его значение создаёт маску вода для поля Input или 
8.  placeholder -- свойство полей, которое подсвечивает серым функцию поля перед тем как в него вводят
                    какую-либо информацию
9.  <hr/> -- тег, который выводит на весь экран горизонтальную линию подчёркивания
10. <select></select> -- тег выводящий на экран выпадающий список
11. <option></option> -- тег создающий опции скажем в выпадающем списке и не только в нём
12. <button></button>:
        1) onclick = свойство кнопки, которое говорит какое действие будет произведено, когда
                        произойдёт событие одиночного клика по кнопке
        2)  ondblclick = тоже самое только событие происходит при двойном клике по кнопке
