I. Теги для изменения стиля текста без CSS:
    1. <strong></strong> -- выделить жирным слово или часть текста
    2. <em></em> -- выделить наклоным шрифтом
    3.  <del></del> -- зачеркнуть по середине текста
    4.  <mark></mark> -- выделить часть текста определённым цветом(по умолчанию жёлтым)
    5.  <span></span>   --  можно предать тексту заключённому в этот тег любой стиль определив ему какой-либо идентификатор
    6.  <q></q> -- берёт в кавычки цитируемое предложение 
    7.  <blockuote></blockuote> --  цитируемую часть текста немного сдвигает
    8.  <small></small> --  текст будет немного уменьшен
    9.  <sub></sub> --  текст будет написан немного внизу
    10. <sup></sup> --  текст будет написан немного вверху
    11. <ins></ins> --  делает нижнее подчёркивание
    12. <hr> -- делает горизонтальную линию на весь экран
    13. <form> </form> -- документ
    14. <label></label> -- название поля
    15. <select></select> -- выпадающий список
    16. <option></option>   -- опции выборы (к примеру в выподающем списке)
    17. <fieldset></fieldset>
    18. <legend></legend>
    19. <textarea></textarea> -- поле 

II. Тег картинки:
    1. Атрибуты:
        1) alt -- если картинка не загрузилась, то обозначает тему картинки (что тут должно быть)
        2)  title -- текст, который возникает при наводе указателя мышки на картинку
    2.  Свойства:
            border-radius: ...% -- округляет картинку

III. Ссылочные теги:
    1.  <a></a> -- тег, который превращает содержимое между скобками в ссылку:
        1) Аттрибуты:
            а.  href="":
                a) ссылку на сайт или на другую страницу сайта
                b)  "mailto:examle@gmail.com" -- ccылка на эл.почту
                c)  "tel:number_tel" -- номер телефона
            b.  target="":
                a)  _blank -- открытие ссылку в другом окне
                b)  _self   --  открытие ссылки в том же окне

IV. Теги HTML для таблицы:
    1.  <table></table> --  рамочный тег таблицы, в котором прописывается вид и стиль таблицы:
        1)  Аттрибуты:
            border=''   --  определяет размер внешней границы таблицы, а также даёт тонкое разграничение внутри, которое не меняется с изменением значения border
            cellspacing="" -- расстояние между внутренними клетками таблицы. Если дать значение 0, то границы ячеек будут общими.
            cellpadding='' -- отступ содержимого ячейки от её границы
            width="" -- ширина таблицы по отношению к ёё мат.тегу
            bgcolor="" -- цвет фона таблицы
    2.  <caption></caption> -- заголовок таблицы, который всегда распологается под тегом <table></table> и над первым тегом <tr></tr>(над первой строкой таблицы)
    3.  <colgroup></colgroup> -- ститлизует столбцы таблицы при помощи её внутреннего <col/> и распологается сразу под тегом <table></table>:
        1) Каждый тег <col/> cоответствует своему столбцу, т.е. первый тег => первому столбцу, воторой тег <col/> => второму столбцу и так далее....
            а)  если использовать его аттрибут span="" и дать ему цифровое значение, то стилизованы будут к-во столбцов таблицы, соответсвующее заданому значению.
    4.  <tr></tr>       --  задаёт строки в таблице  
    5.  <td></td>       --  пишется внутри тега <tr></tr> и определяет столбец с названием или содержимым
    6.  <thead> -- оборачивает <tr></tr>, которая содержит заголовки таблицы:
        1)  В этом случае внутри используем как внутренний тег <tr></tr> не тег <tr></tr> а тег <th></th>, который автоматом стилизует иначе шрифт заголовков
    7.  <tfoot></tfoot> --  используем в случае если есть нужда обернуть нижние строки или строку таблицы допустим сделать из последней строки строку итогов.
                            Для этого в <td></td> тега <tfoot></tfoot> используем атрибут colspan="", который объеденяет вместе в одной строке то к-во столбцов,
                            которое указывается в нём как значение
    8.  <tbody></tbody> --  в нём содержится основное содержимое таблицы.

V.  Формы:
    1. <form></form> -- тег для оборачивания содержимого формы
        1) Аттрибуты:
            a.  action="" -- его значение говорит, куда или кому передавать данные формы
            b.  method="" -- 
            с.  оninput="" -- это событие по случаю которого (что-то введено) выполняется код, написанный в кавычках
    1. <label></label> -- содержит название поля к которому принадлежит по идентификатору:
        1) Аттрибуты:
            a.  for="name_class", поля которому данный тег принадлежит
    2. <input/> -- поле служащее для ввода данный пользователем:
        1) Аттрибуты:
            а.  type="" -- значение определяет тип/форму вводимых данных:
                a)  "radio" -- превращает поле input в радиобокс
                    -- если есть несколько радиобокс в форме, то чтобы мы не могли выделить сражу несколько из них нужно
                        использовать аттрибут name=''(подобно идентификатору), если задать нескольким радиобоксам одно и тоже name
                        то тогда возможно будет выбрать только один радиобокс из всей этой группы
                    -- в радиобоксах всегда нужно также определять атрибут value="", чтобы  было понятно какую информацию
                        нужно передать при выборе данного радиобокса.
                b)  submit -- превращает input в кнопку передающую данные введённые в поля формы у который есть идентификатор типа name
                с)  сheckbox -- 
                d)  color -- даёт выбрать цвет
                е)  text -- говорит, что будет введён текст
                f)  range -- это бегунок:
                    --  Аттрибуты: 
                        ... min="0" max="100" границы бегунка
            b.  placeholder -- текст, который подсвечивается серым, чтобы дать понять пользователю, какую информацию ему нужно ввести в данном поле.
            с.  value="" -- текстовое значение, которое будет написано в поле при загрузке формы на лист (значение по умолчанию)
            d.  list="" -- атрибут, который связан только с внутреним тегом тега input, который называется <datalist></datalist>
                            Он связывается с ним по значение атрибута list и по названию класса тега <datalist></datalist>
    3.  <fieldset></fieldset>   -- оборачивает теги формы и ввыводит их на лист с группироваными в отдельную рамку (как табличка с полями):
        1)  <legend></legend> --    указывается сразу же под тегом <fieldset></fieldset> и задаёт этой рамке название
    4.  <select></select> --   выпадающий список:
        1)  <option></option> --  внутри пишется опция для выпадающего списка
            а.  Аттрибуты:
                а)  value="" -- значение говорит какую информацию переслать при выборе данной опции 
    5.  <textarea></textarea>  -- выводит текст в готовой рамке, а также позволяет вписать свой текст, но только в границах определённой самой рамкой:
        а.  Аттрибуты:
                а)  rows="" -- значение определяет к-во строк в рамке
                b)  cols="" -- значение определяет к-во столбцов в рамке
    6.  <datalist></datalist> -- очень похож на выпадающий список, но он является внутреним тегом тега input и требует связи с ним посредствам 
                                    атрибута тега input такого как list и идентификатору тега <datalist></datalist>. Ещё одно отличие этого поля отдельную
                                    поля select, что можно в нём написать какой то символ и он покажет выпадающий список уже отфильтрованый по этому символу
    7.  <output></output>  --   тег используемый для вывода информации на экран. Может быть связан по средствам JS с тегом input типа range (бегунок)

VI. Кнопки <button></button>:

            а)  submit -- передача данных определённых кодом JS или в атрибуте action=""
            b)  reset --  обнуление значений формы
